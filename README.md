# ByeDPI iOS

iOS приложение для обхода DPI (Deep Packet Inspection) блокировок, аналог Android версии ByeDPI.

## Описание

Это приложение создает VPN туннель для перенаправления всего сетевого трафика через специальные механизмы обхода DPI блокировок провайдеров.

## Особенности

- ✅ Создание VPN туннеля без внешнего сервера
- ✅ Обход DPI блокировок на уровне пакетов
- ✅ Настраиваемые методы обхода (фрагментация TCP, модификация заголовков и др.)
- ✅ Интуитивный интерфейс на SwiftUI
- ✅ Настройки сохраняются между сессиями
- ✅ Поддержка iPhone и iPad

## Требования

- iOS 17.0 или новее
- Устройство с поддержкой Network Extensions
- Apple Developer Account для подписи (для установки на устройство)

## Установка

1. Откройте проект в Xcode
2. Выберите таргет "ByeDPI-iOS"
3. Выберите ваше устройство или симулятор
4. Нажмите "Run" (Cmd+R)

## Настройка проекта

### Для разработки:
1. Замените Bundle Identifier на ваш собственный
2. Настройте App Groups в Capabilities
3. Убедитесь, что Network Extensions capability включены

### Для распространения:
1. Создайте App ID с Network Extensions capability
2. Создайте Provisioning Profile
3. Подпишите оба таргета (основное приложение и Network Extension)

## Использование

1. Запустите приложение
2. Нажмите "Подключить VPN" для активации туннеля
3. Статус изменится на "Подключено" при успешном запуске
4. Перейдите в "Настройки" для настройки параметров обхода

## Методы обхода DPI

Приложение поддерживает несколько методов обхода:

1. **TCP Fragmentation** - Разбиение TCP пакетов
2. **HTTP Header Modification** - Модификация HTTP заголовков  
3. **TLS Padding** - Добавление случайных данных в TLS пакеты
4. **Random Techniques** - Комбинация различных методов

## Архитектура

Проект состоит из двух основных компонентов:

### Основное приложение (ByeDPI-iOS)
- Интерфейс пользователя (SwiftUI)
- Управление VPN соединением
- Настройки и конфигурация

### Network Extension (ByeDPI-Packet-Tunnel)
- Обработка сетевых пакетов
- Реализация методов обхода DPI
- Управление туннелем
